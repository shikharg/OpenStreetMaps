Database Instructions

1. To set up a database for running this program, make sure you have an instance of MySQL running on your computer or server.

2. Secondly, make sure you have write permissions.

3. Head to SQLFluxCapacitor.java class LINE 35 inside the project.projectfiles package, 
and ensure that your SQL server user and password match those entered.

4. Next, run the included file database_setup.sql.

Starting the Application

1. Go to the project.projectfiles package.

2. Run the main class, IntelligentTrafficApplication.java
